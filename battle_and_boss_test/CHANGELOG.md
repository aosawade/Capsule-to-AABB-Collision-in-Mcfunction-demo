# 更新日志

本文档记录了 Battle & Boss Test 数据包的所有版本更新。

## 版本约定

使用语义化版本号：`主版本.次版本.修订版本`

- **主版本**：不兼容的 API 修改或重大功能更新
- **次版本**：向下兼容的功能性新增
- **修订版本**：向下兼容的问题修复

## [1.0.0] - 2026-02-17

### 新增功能
- 实现相对坐标胶囊体 ↔ AABB 碰撞检测系统
- 添加玩家跳跃下半身无敌系统
- 添加玩家闪避无敌系统
- 集成 Animated Java 动画支持
- 添加碰撞箱可视化调试工具
- 添加完整的测试函数套件
- 创建全面的文档系统

### 技术实现
- 基于 Slab Method 的射线-AABB 相交算法
- 相对坐标系统减少计算开销
- 模块化架构设计
- 性能优化（快速排除、距离筛选）
- 与 Minecraft 1.21.11 完全兼容

### 文档
- 创建 README.md 项目说明文档
- 创建 API_REFERENCE.md 详细接口文档
- 创建 LICENSE MIT 许可证
- 创建 CHANGELOG.md 更新日志
- 创建 CONTRIBUTING.md 贡献指南

### 已知问题
- 仅支持主世界（overworld）碰撞检测
- 胶囊体数量过多可能影响性能
- 需要玩家手动添加 `aabb` 标签

### 依赖项
- Minecraft 1.21.11+（pack format 94）
- 无外部数据包依赖
- 可选：Animated Java 用于动画集成

## 未来计划

### 版本 1.1.0（计划中）
- 多世界碰撞检测支持
- 更高效的胶囊体筛选算法
- 碰撞事件回调系统

### 版本 2.0.0（规划中）

- 可视化编辑器工具

## 迁移指南

### 从旧版本迁移
当前版本为初始版本 1.0.0，无迁移需求。

### 向后兼容性
所有 API 在次版本更新中保持向下兼容。主版本更新可能包含破坏性更改。
